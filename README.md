# 2^3 Factorial Design Calculator 🧪

A lightweight Python script to calculate Main Effects and Interaction Effects ($A, B, C, AB, AC, BC, ABC$) for a $2^3$ factorial experiment.

## 📋 Features
* **Calculates all effects:** Automatically computes 3 main effects and 4 interaction effects.
* **Input Validation:** Handles decimal inputs and prevents crashes from non-numeric entries.
* **Clean Output:** Formats results to 4 decimal places for scientific accuracy.
